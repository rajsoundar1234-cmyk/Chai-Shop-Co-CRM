# Chai-Shop-Co-CRM

This is a Salesforce CRM application developed for TheChaiShop&Co.

## Features
- Customer, Employer, Enterprise Management
- Automated Billing (Order Quantity & Total Bill)
- Validation Rules (Aadhar, PAN, Phone)
- Email Automation (Birthday Offers)
- Reports & Dashboards

## Technologies Used
- Salesforce Lightning
- Flows
- Reports & Dashboards
